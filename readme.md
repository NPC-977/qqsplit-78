# qqsplit_78

*A short description of the keyboard/project*

* Keyboard Maintainer: [qqsplit_78](https://github.com/NPC-977)

* 支持vial动态改键，fn层、大写指示灯。
* fn层指示灯支持修改颜色(默认为fn层的左fn、fn层的右fn，支持在vial内修改)。
* 自定义的rgb关闭键(关闭后不影响大写指示灯，支持在vial内修改)
